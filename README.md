## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

```bash
npm run dev
# or
yarn dev
```

Clone the repo

```sh
git clone https://github.com/UniversallyRy/starWars-wiki.git
```

Install NPM packages

```sh
yarn
```
